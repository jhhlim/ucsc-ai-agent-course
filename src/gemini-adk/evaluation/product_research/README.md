## Product Research simple example

This example illustrates ambiguous instructions about which tool to use when looking up production info.

### Example Prompt to use
I am looking for information about the headphones in the catalog

### Update the response in the eval set to
The available headphones are Wireless headphones with advanced noise cancellation technology for immersive audio

### Model
* By default this example will use a model from Doubleword

### LangSmith Integration
* If you would like to enable tracking, make sure to set LANGSMITH_TRACING to true and provide your LANGSMITH_API_KEY
